# python-24fa-academic-project
This repository contains a collection of Python applications during my learning journey from October 2024, demonstrating core competencies in automation, data structures, and algorithmic logic.
## Project Breakdowns

### 1. Automation & Office Document Generation (`A4_Part1.py`)
* **Description:** An automation script that reads student data from a matrix spreadsheet (`.xlsx`) and dynamically generates customized Word certificates.
* **Key Skills:** External library integration (`openpyxl`, `docxtpl`), exception handling, data type verification, and automated reporting.

### 2. Algorithmic Currency Breakdown System (`A3_Part3.py`)
* **Description:** An optimization script that calculates the minimum number of currency notes/denominations required for a given financial amount. 
* **Key Skills:** Matrix/list formatting, floor division (`//`) and modulo (`%`) mathematical logic, and error-auditing loop controls (`try-except`).

### 3. Data Structure Manipulation (`A2_Part2.py`)
* **Description:** A program that collects multiple numerical data samples from a user, merges the datasets, sorts them, and filters out duplicate items.
* **Key Skills:** Array/list operations (`.append()`, `.extend()`, `.sort()`), type casting, and working with unique collections (`set`).

### 4. Interactive Console Grade Processor (`A2_Part1.py`)
* **Description:** A console-based utility that processes continuous student score entries, checks numerical boundaries, and assigns academic grades.
* **Key Skills:** Infinite control loops (`while True`), complex conditional branching (`if/elif/else`), and user session control.
